# VIN Decoder

Це веб-застосунок для розшифрування VIN-кодів (Vehicle Identification Number) транспортних засобів. Він дозволяє швидко отримати базову інформацію про автомобіль за його унікальним 17-символьним ідентифікатором.

## Опис функціоналу

Застосунок надає наступні можливості:

- **Введення VIN-коду**: Користувач може ввести 17-значний VIN-код у спеціальне поле.
- **Валідація**: Перевіряється коректність введеного коду (довжина та допустимі символи).
- **Отримання даних**: Після надсилання форми, застосунок виконує запит до публічного API ([NHTSA vPIC](https://vpic.nhtsa.dot.gov/api/)) для отримання даних про транспортний засіб.
- **Відображення результату**: Отримана інформація (виробник, модель, рік випуску, тип кузова тощо) відображається у вигляді зручного списку або карток.
- **Обробка помилок**: У разі відсутності даних або помилки запиту, користувач отримує відповідне повідомлення.
- **Адаптивний дизайн**: Інтерфейс оптимізовано для перегляду на різних пристроях (від мобільних телефонів до настільних комп'ютерів).
